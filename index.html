📦 NEWNEW ORIGINS Maven Repository

📝 说明: 这是 newnew origins 模组的 Maven 仓库，包含所有发布版本。

🚀 开发者

在 build.gradle 中添加：

repositories {
    maven {
        name = 'NEWNEW ORIGINS'
        url = 'https://gam0zing.github.io/newnew_origins-maven/releases/'
    }
}

dependencies {
    implementation 'com.gam0zing:newnew_origins:1.0.0'
}

📋 可用版本

1.0.0 最新
